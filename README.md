# BOSS 岗位粗筛助手

一个本地 Chrome 扩展，用来在 BOSS 直聘岗位列表页做粗筛、收藏、记录和复核。

它适合先把明显不合适的岗位过滤掉，把可能合适的岗位收藏或放进待复核，再导出表格继续人工判断。

## 边界

这个扩展只做：

- 岗位粗筛
- 自动收藏达到分数线的岗位
- 跳过明确已处理的岗位
- 记录筛选结果
- 导出表格 / JSON
- 导出诊断信息

这个扩展不做：

- 自动打招呼
- 自动投递
- 读取聊天记录
- 绕过验证码或账号限制
- 多账号、代理池、隐藏浏览器等操作

遇到验证码、登录异常、账号安全提示时，应立即停止。

## 主要功能

- 按岗位性质和目标方向打分
- 支持薪资范围过滤
- 支持招聘者活跃度过滤
- 支持跳过历史明确排除 / 已收藏岗位
- 支持待复核岗位随规则更新重新判断
- 支持多关键词扫描
- 支持复制表格、JSON 和诊断信息
- 可选 DeepSeek 生成搜索词和筛选规则

DeepSeek API Key 不会内置在代码里。使用者如果需要 AI 生成规则，需要自己在扩展面板里填写 Key。

## 安装

1. 下载或克隆本仓库。
2. 打开 Chrome：`chrome://extensions/`
3. 打开右上角「开发者模式」。
4. 点击「加载已解压的扩展程序」。
5. 选择本仓库文件夹。
6. 打开 BOSS 直聘岗位列表页。

匹配页面：

```text
https://www.zhipin.com/web/geek/jobs*
```

## 使用

1. 先在 BOSS 页面手动设置城市、薪资、公司规模等条件。
2. 在扩展面板里填写岗位性质，例如 `销售`。
3. 填写目标方向，例如 `AI客服`、`AI应用`、`SaaS`。
4. 检查正向词和反向词。
5. 选择当前列表或多关键词扫描。
6. 点击开始。
7. 扫描后复制表格或 JSON 做二次复核。

## 当前规则重点

- 招聘者超过 3 日未活跃的岗位会排除。
- `在线`、`今日活跃`、`3日内活跃` 不因活跃度排除。
- 日薪 / 时薪岗位会排除。
- 多段薪资会优先看和目标薪资有交集的一段。
- 历史 `已排除`、`已收藏`、`用户不想看` 会直接跳过。
- 历史 `待复核`、`收藏失败`、`用户想看` 会允许重新判断。

## 文件说明

```text
manifest.json   Chrome 扩展声明
content.js      页面面板、岗位识别、打分和收藏逻辑
background.js   DeepSeek 请求转发
```

## 隐私和安全

- 扩展主要在当前 BOSS 页面本地运行。
- 扫描结果保存在浏览器本地存储里。
- DeepSeek Key 由使用者自己填写，代码仓库不包含任何 Key。
- 不建议长时间无人值守运行。
