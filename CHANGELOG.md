# 更新记录

## 0.1.4 / rough-screen-v17 - 2026-06-11

- 收藏标准改为：宽松 50、标准 60、严谨 70。
- 公开包默认不带个人岗位偏好词，避免别人安装后继承“销售/AI应用”的筛选倾向。
- 本地生成排除词时不再默认排除保险、贷款、课程销售，是否排除由用户自己填写或由 AI 按当前目标生成。

## 0.1.3 / rough-screen-v16 - 2026-06-10

- “DeepSeek 生成”改为“AI 生成”。
- 新增服务商选择：DeepSeek、通义千问、智谱 GLM、Kimi、硅基流动。
- 后台请求按服务商切换 API 地址，模型名仍可手动修改。
- 扩展权限增加对应国产 AI 服务商 API 域名。

## 0.1.2 / rough-screen-v15 - 2026-06-10

- 本次扫描确认不再展示具体收藏分数，收藏标准由宽松/标准/严格按钮控制。
- “直接排除”改为“本次避开”，避免把用户当前方向的避开词写成通用规则。
- 明确提示避开词只服务于当前目标方向，不代表所有用户都应该排除。

## 0.1.1 / rough-screen-v14 - 2026-06-10

- 面板标题显示插件版本和规则版本。
- “配置摘要 / 规则预览”改为“本次扫描确认”。
- 配置说明压缩成当前目标、处理方式、直接排除和安全边界。
- 复制配置提示改为排查问题时使用，避免误导成产品说明书。

## 0.1.0 / rough-screen-v13 - 2026-06-10

- 首次开源版本。
- 支持岗位粗筛、自动收藏、记录、复制表格和多关键词扫描。
