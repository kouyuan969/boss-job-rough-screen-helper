怎么安装

下载后加载到 Chrome

  1. 下载最新版 zip，并先解压。
  2. 打开 Chrome，进入 chrome://extensions/。
  3. 打开右上角「开发者模式」。
  4. 点击「加载已解压的扩展程序」。
  5. 选择解压后的插件文件夹。
  6. 打开 BOSS 直聘岗位列表页，右侧出现助手面板就能用。
