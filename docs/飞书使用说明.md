# BOSS 岗位粗筛助手：快速使用说明

这是一个 Chrome 扩展，用来在 BOSS 直聘岗位列表页做岗位粗筛。它会按你填的岗位性质、目标方向、薪资和排除词判断岗位，匹配度高的岗位可以自动收藏，并把结果记录成表格。

它只做粗筛、收藏、记录和复核；不自动打招呼，不自动投递，也不读取聊天内容。

## 下载

GitHub 地址：

https://github.com/kouyuan969/boss-job-rough-screen-helper

不会用 Git 的话，打开页面后点 `Code` -> `Download ZIP`，下载后先解压。

## 安装

1. 打开 Chrome，进入 `chrome://extensions/`。
2. 打开右上角「开发者模式」。
3. 点击「加载已解压的扩展程序」。
4. 选择解压后的插件文件夹。
5. 打开 BOSS 直聘岗位列表页：`https://www.zhipin.com/web/geek/jobs`

页面右侧出现「BOSS 岗位粗筛助手」面板，就说明安装成功。

## 怎么用

1. 先在 BOSS 页面选好城市、薪资、公司规模等条件。
2. 在插件里填写岗位性质，比如：`销售`、`BD`、`客户成功`、`售前`。
3. 填写目标方向，比如：`AI客服`、`AI应用`、`AI SaaS`、`智能客服`。
4. 检查正向词和排除词，先别写太多。
5. 第一次建议只扫 10-20 个岗位，确认规则没跑偏后再扩大数量。
6. 扫完后点「复制表格」，粘到飞书表格或 Excel 里复查。

## 结果怎么看

- `已收藏`：匹配度较高，插件已经尝试收藏。
- `待复核`：有点相关，但需要人工再看。
- `已排除`：明显不合适，一般不用看。
- `收藏失败`：岗位可能合适，但页面按钮没点成功，可以手动确认。

优先看 `已收藏` 和 `待复核`。

## 使用提醒

- 目标方向不要只写 `AI`、`软件`、`互联网`，尽量写到具体场景。
- 招聘者超过 3 日未活跃的岗位会排除。
- 日薪、时薪岗位会排除。
- 已排除、已收藏、用户不想看的岗位会跳过。
- 看到验证码、账号异常、频率提示时，立即停止。
- 不建议长时间无人值守运行。

DeepSeek API Key 不是必填。不填也能用本地规则；如果要用 AI 生成搜索词和规则，需要使用者自己填写 Key。
